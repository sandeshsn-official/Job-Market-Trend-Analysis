# Job Market Trend Analysis and Skill Gap Prediction

## 📌 Overview

This project analyzes job market trends by collecting job postings, preprocessing text data, extracting in-demand skills, performing skill gap analysis, and predicting future job trends using machine learning.

## 🚀 Features

- Job data scraping
- Data preprocessing
- Skill extraction
- Exploratory Data Analysis (EDA)
- Skill Gap Analysis
- Trend Prediction
- Interactive Streamlit Dashboard

## 🛠 Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Plotly
- BeautifulSoup
- Selenium

## 📂 Project Structure

JobMarketAnalysis/

## 📊 Machine Learning Model

- Random Forest Regressor
- TF-IDF Vectorizer
- Label Encoding
- Feature Engineering

## 📈 Results

- Identified high-demand skills
- Performed skill gap analysis
- Predicted future hiring trends
- Interactive dashboard for visualization

## ▶️ Installation

git clone https://github.com/sandeshsn-official/Job-Market-Trend-Analysis.git

cd Job-Market-Trend-Analysis

pip install -r requirements.txt
